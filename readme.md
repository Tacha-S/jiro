##使い方

###Toppingクラス

#####基本的な使い方
トッピングクラスのインスタンスを生成します。  
イニシャライザにコールの文字列を入れてください。
  
    Topping topping = new Topping("ニンニク抜きヤサイチョモランマアブラマシマシカラメ");
    
このメソッドでコンソールに出力を行います

    topping.outputToppingInfo();
以下コンソール出力です。

    【ニンニク抜きヤサイチョモランマアブラマシマシカラメ】
    ニンニク：抜き
    アブラ：マシマシ
    ヤサイ：チョモランマ
    カラメ：デフォルト
